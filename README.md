# Sound-API-Lab-1
Design and implement a custom audio control interface using HTML and JavaScript. Your interface must allow a user to interact with an audio file through programmatic controls.
Minimum Requirements (Core Functionality) - 25 marks
Your application must include:
1. Audio Playback (10 marks)
Load an audio file; Implement Play and Pause button
2. Volume Control (5 marks)
Allow the user to: Increase/decrease volume OR use a slider
3. One Additional Feature (10 marks)
Choose at least ONE:
Mute / Unmute
Loop / Repeat
Playback speed control
Display current playback time
Skip forward/backward
Creativity & API Usage - 20 marks
Marks here are awarded for how you use the Sound API, not the audio content.
You may include at least one of these:
Custom UI controls (buttons, layout, styling)
Multiple audio controls working together
Interactive behaviour (e.g., auto-play triggers, dynamic changes)
Unique ways of controlling playback (e.g., keyboard controls, events)
GitHub Submission - 5 marks
